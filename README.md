🎴 Card Matching Game (2 Players)
A classic 2-player memory card matching game built using pure HTML, CSS, and JavaScript. Players take turns flipping over cards on a grid to find matching pairs and score points.

🎮 How the Game Works
The objective of the game is to find more matching pairs of cards than your opponent.
1. Setup:
A grid of face-down cards is displayed.
Player 1 starts the game.

2. Turn Mechanics:
On your turn, click to reveal 2 cards.
Match: If the designs on both cards match:
You earn 1 point.
The matched cards remain face-up.
You get another turn!
Mismatch: If the cards do not match:
Both cards are flipped back face-down after a brief delay.
Turn switches to the other player.

3. Winning the Game:
The game continues until all card pairs are matched.
The player with the highest score at the end wins!

🛠️ Built With
HTML5 – Game structure, grid container, and board layout.
CSS3 – Styling, responsive card grid design, and card flip animations.
JavaScript (Vanilla) – Game logic, state management, turn-switching, score tracking, and matching validation.

⚡ Features
2-Player Local Turn-Based Gameplay – Play with a friend on the same screen.
Dynamic Score Board – Keeps track of each player's matches and highlights whose turn it is.
Card Flip Animations – Smooth CSS animations when revealing or hiding cards.
Randomized Grid Shuffle – Cards are shuffled randomly every time you start or restart a game.
